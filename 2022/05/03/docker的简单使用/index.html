发表于 2022-05-03 更新于 2022-10-10

kali 中安装 docker

apt安装

docker官网并没有在kali上面安装docker的指南，但是有 debian的，不过在kali上有点小问题。直接用apt装，用一段时间再说。

使用apt搜索docker开头的软件包，有个包就叫docker，但装完之后没什么反应，docker.io应该是最重要的，还能找到一些插件，比如docker-compose

docker官网指南，在debian上安装docker

docker官网有给安装的命令 https://docs.docker.com/engine/install/debian/（kali是基于debian的）

docker官网上，把docker，docker.io等 称为 old versions，所以要先删除旧版本

然后设置仓库，应该指的是apt的仓库。update更新的是apt的索引，upgrade是更新软件，有些软件包kali应该已经安装过了

然后获取公钥

然后就是出问题的一步

lsb_release -cs 运行结果应该是 debian的版本名，比如buster、stretch这种，但是在 kali 上，运行结果是kali-rolling 所以就出问题了

在kali中查看debian的版本

换成对应的版本名字应该就可以了，然后用apt安装docker

没有真正运行这种方法，到出问题的那一步寄了，所以直接安装了docker.io

一些小问题

docker.io 说是旧版本，但我看他版本号挺高的，网上找到一种说法，说是docker.io是debian团队维护的，docker-ce是docker官方维护的

apt中查看docker.io的信息，可以看到维护团队的信息

为什么不在windows上装

windows家庭版不会开hyper-v

以前hyper-v好像和vmware冲突，只能用选一个，vmware比较重要。但是网上有消息说现在已经解决了
